Apple Pie Game

The Apple Pie Game is a simple word-guessing game from the "Develop in Swift Fundamentals". Players guess a word by submitting letters, with limited attempts for incorrect guesses. The game uses Swift and to teach key programming concepts such as user input handling, game logic, and user interface design. The goal is to guess the word before running out of attempts. It introduces concepts like arrays, string manipulation, and conditionals while building a basic, interactive game.

This project highlights my proficiency in Python for data manipulation, SQL for database management, and integrating multiple database systems to create an efficient data workflow.
